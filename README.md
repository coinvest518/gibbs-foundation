# The Gibby Foundation

A beautiful, responsive website for The Gibby Foundation, a non-profit organization providing youth basketball programs and community support.

## About

This website showcases the foundation's programs, impact, events, and donation opportunities. Built with modern web technologies for optimal performance and accessibility.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Dark Theme**: Modern dark UI with Tailwind CSS
- **Interactive Elements**: Mobile navigation, donation form, and animated components
- **Accessibility**: Proper semantic HTML and ARIA labels
- **Fast Loading**: Optimized with Tailwind CDN and minimal dependencies

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- SVG Icons

## Getting Started

### Option 1: View Locally
1. Clone or download this repository
2. Open `index.html` in your web browser

### Option 2: Deploy to GitHub Pages
1. Create a new repository on GitHub
2. Push this code to the repository
3. Go to Settings > Pages
4. Select "Deploy from a branch"
5. Choose "main" branch and "/ (root)" folder
6. Click Save

Your site will be live at `https://yourusername.github.io/repository-name/`
